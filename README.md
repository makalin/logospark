# **LogoSpark**

An AI-powered web application to brainstorm project names, generate unique logos, and customize them in real-time.

## **The Problem**

Coming up with a great brand identity is hard. You need a catchy name and a professional logo that fits your vision. Traditional logo generators are often template-based and lack originality, while working with a designer is time-consuming and expensive.

## **The Solution**

**LogoSpark** is an all-in-one identity toolkit. It uses generative AI to help you from the very first step (thinking of a name) to the last (exporting a production-ready logo).

It's built on the idea of "AI-assisted creation," where the AI provides the creative spark, and you provide the final direction and customization.

## **Tech Stack**

* **Frontend:** React (Vite)  
* **Styling:** Tailwind CSS  
* **AI (Images):** Google's Imagen 3 API  
* **AI (Text):** Google's Gemini 2.5 Flash API  
* **Canvas (Future):** fabric.js for advanced vector manipulation.
